This package is a free version of the ‘Animal Rug’ package and can be used freely as you wish.

The materials are Unity’s own standard materials. Therefore, you can use them in every version of Unity, including Unity 6.

The demo scene was built in URP. To use it in Built-in or HDRP, you need to convert the materials.

Downscale the textures for mobile

Note: Materials use the transparent mode for transparency. However, the transparent option can cause problems in many cases during rendering, especially with fog. This is a limitation not only in Unity but in all game engines in general. Handling alpha is always tricky. 
If you encounter such an issue, try using the "Alpha Clip" mode. This will solve your problem, but it may still cause minor artifacts.
